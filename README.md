# Central Bank of Egypt Currency Exchange Rates Kiosk

Fullscreen Raspberry Pi kiosk displaying official Central Bank of Egypt exchange rates. Updates every 15 minutes.
## Install (one line)
```bash
curl -fsSL https://raw.githubusercontent.com/omarthegeek/cbe-rates-kiosk/refs/heads/main/install.sh | bash
```

Then reboot.

## Uninstall
```bash
curl -fsSL https://raw.githubusercontent.com/omarthegeek/cbe-rates-kiosk/refs/heads/main/uninstall.sh | bash
```

## Requirements
- Raspberry Pi OS
- Internet connection on boot
